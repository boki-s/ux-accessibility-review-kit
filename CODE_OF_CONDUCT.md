# Code of Conduct

This project is intended to be welcoming to designers, developers, students, educators, and accessibility learners.

## Expected Behavior

- Be respectful and constructive
- Explain suggestions clearly
- Assume people are learning
- Focus feedback on the work, not the person
- Help keep accessibility conversations practical and inclusive

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Dismissive comments about people learning accessibility
- Discriminatory language or behavior
- Repeated off-topic or disruptive comments

## Reporting

If you see behavior that makes the project less welcoming, open a GitHub issue or contact the maintainer through the public profile links.

Maintainers may remove comments, close issues, or block participants when needed to protect the project community.
