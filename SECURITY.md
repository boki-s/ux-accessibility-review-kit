# Security Policy

UX Accessibility Review Kit is a static HTML, CSS, and JavaScript project. It does not collect user data, use a backend, or send review progress to a server.

## Reporting a Concern

If you notice a security or privacy concern, please open an issue with:

- A clear description of the problem
- Steps to reproduce it
- The browser and operating system used
- Any suggested fix, if available

Please do not include sensitive personal information in public issues.
